Neighborhood App

This application was created completely from scratch. The idea comes from a prompt at the end of an Udacity course I took linked here: Intro to Javascript Design Patterns

It is a simple app that lets you search for "Interesting" locations near you, using Yelp APIs, like restaurants, barbershops, gas stations, etc. You can find my github repo here: https://github.com/aruballo/neighborhoodapp
